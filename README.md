# Motion
📔 이미지, 동영상, 메모, 투두리스트를 추가, 삭제할 수 있는 메모장

배운 것
- typescript의 장점인 type과 interface를 사용해 객체지향적 프로그래밍
- 생성자를 이용한 의존성 주입
- draggable api

### 추후 보강 계획
- decorator를 사용해 동적으로 composition
